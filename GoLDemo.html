Conway's Game of Life

Conway's game of life is a 0 player 'game', that, using a set of rules, evolves the starting board (given in a text file) and gives us an animation output

Here is a set of starting boards and their resulting animation.

Conway's rules are:

  • A cell lives if it has 2, or 3 live neighbours
  • A cell 'breeds' if it has 3 live neighbours
  • Otherwise the cell dies
